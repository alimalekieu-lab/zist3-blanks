# تمرین و تسلطِ زبان انگلیسی ۳ (Vision 3)

اپلیکیشنِ جامعِ آماده‌شدن برای **امتحان نهاییِ زبان انگلیسی ۳** (پایهٔ دوازدهم) — دارک‌مود، مناسبِ تمرکز، با ذخیرهٔ خودکارِ پیشرفت.

## فایل‌ها
- **`index.html`** — نقشهٔ راهِ ۲۰: کلِ کتاب + کتاب کار + آزمون‌های نهایی. شاملِ واژگان (با شمارندهٔ تکرار و حافظهٔ تصویری)، گرامرِ کامل با نمای یک‌نگاهی، کلماتِ خارج از کتاب 🚩، پیش‌بینیِ امتحان بعدی، بانکِ تعریف، ضرب‌المثل‌ها، مرورِ Listening، تکنیکِ حلِ هر سؤال، و همهٔ کلماتِ متنِ ریدینگ‌ها. با چک‌لیستِ تیک‌زدنی و تایمرِ مطالعه.
- **`final-exams.html`** — تمرینِ تعاملیِ سه آزمونِ نهاییِ واقعی (خرداد ۱۴۰۲، ۱۴۰۳، ۱۴۰۴) با تصحیحِ خودکار طبقِ کلیدِ رسمی.

## نکته
بخشِ Listening نیاز به فایلِ صوتیِ رسمی دارد (از `azmoon.medu.ir`)؛ محتوا و کلیدِ آن در اپ آمده است.
