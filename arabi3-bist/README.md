## چه چیزی داخلش هست؟

| بخش | تعداد |
|---|---|
| بخش‌های قاعده و نکته | ۱۴ |
| کارت‌های تمرین (واژه، جمله، ترجمه) | ۸۲۰ |
| مجموع بخش‌ها | ۳۰ |

محتوا از **متن، تمارین، معجم و پاورقی‌های هر چهار درس** کتاب درسی استخراج شده است.

- **قواعد پایه:** ضمیرها و ۱۴ صیغه، صرف کامل ماضی/مضارع/امر/نهی، ناصبه و جازمه، مجهول، کان، هشت باب ثلاثی مزید و مشتقات، نقش‌ها و اعراب
- **قواعد چهار درس:** حروف مشبهه و چهار «لا» • الحال • الاستثناء • المفعول المطلق
- **جدول طلایی:** ۲۴ صورت فعل با ترجمهٔ دقیق فارسی
- **واژگان:** معجم چهار درس + پاورقی‌ها + مترادف و متضاد + مفرد و جمع + تعریف‌ها
- **جملات:** کل متن‌های چهار درس، آیات و احادیث، متن‌های علمی، ضرب‌المثل‌ها — همه با ترجمه
- **۲۵ تلهٔ کشنده** و تکنیک درک مطلب
- **برنامهٔ ۴۸ ساعته**

## چطور کار می‌کند؟

هر کارت فقط عربی را نشان می‌دهد و ترجمه‌اش تار است. اول ترجمه را در ذهن بساز، بعد روی بخش تار بزن تا باز شود.

- **✓** = بلدم &nbsp;•&nbsp; **★** = برای مرور نشان کن
- فیلترهای بالای صفحه: همه / هنوز بلد نیستم / ⭐ نشان‌دار / ✓ بلدم
- هر بخش شمارندهٔ «بار خواندم» و نوار پیشرفت خودش را دارد
- حلقهٔ بالای صفحه، درصد کل ۸۲۰ کارت است
- پیشرفت ذخیره می‌شود؛ ببندی و برگردی سرِ جایش است
