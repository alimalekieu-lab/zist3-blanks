# فصل ۷ — فناوری‌های نوین زیستی | کارگاه یادگیری

یک صفحهٔ درسی تعاملی و تک‌فایل برای یادگیری کامل فصل ۷ زیست‌شناسی ۳ (فناوری‌های نوین زیستی).

بدون نیاز به سرور کار می‌کند: هم با دابل‌کلیک روی `index.html` و هم روی **GitHub Pages**.

## محتوا
- هر سه گفتار با تمام جزئیات و واژه‌های کتاب: مهندسی ژنتیک و همسانه‌سازی دنا، مهندسی پروتئین و بافت، یاخته‌های بنیادی، بیوانفورماتیک، و کاربردهای کشاورزی/پزشکی/اقتصادی.
- دیاگرام‌های بازسازی‌شده (SVG) همراه با متن داخل تصویر: برش EcoR۱ و انتهای چسبنده، دیسک و دنای نوترکیب، شوک ورود به میزبان، جداسازی با پادزیست، زنجیرهٔ C انسولین، مراحل ژن‌درمانی و دام تراژن.
- کارت‌های «چرا؟» با پاسخ کشویی برای مرور فعال، و باکس‌های «بیشتر بدانید».

## امکانات
- حالت روشن/تاریک (دارک‌مود) با تم درخشش سبز GFP.
- ابزار یادداشت روی صفحه: قلم، هایلایتر، پاک‌کن، انتخاب رنگ، تنظیم ضخامت، واگرد/ازنو، نمایش/مخفی و پاک‌کردن همه.
- میان‌بر صفحه‌کلید: `P` قلم، `H` هایلایتر، `E` پاک‌کن، `S` خواندن، `Ctrl+Z` / `Ctrl+Y`.

index.html                              ← صفحهٔ کارگاه (همان محتوا)
