# 🐦 رفتارهای جانوران — راهنمای کامل مطالعه

راهنمای تعاملی و رنگیِ **فصل ۸ زیست‌شناسی (رفتارهای جانوران)** برای آمادگی **امتحان نهایی و کنکور**.
تمام محتوای فصل به‌صورت دقیق و مطابق کتاب درسی، همراه با ابزارهای یادگیری فعال، در یک فایل HTML مستقل گردآوری شده است.

**نسخهٔ زنده (بعد از فعال‌کردن GitHub Pages):** `https://<username>.github.io/<repo>/`

---

## ✨ امکانات

- 🗺️ **نقشهٔ مفهومی** کامل هر سه گفتار
- 🎨 **کارت‌های رنگی** برای هر نوع رفتار (کدگذاری رنگی برای تقویت حافظه)
- 🔬 جعبه‌های **آزمایش** و **«بیشتر بدانید»** با تمام جزئیات کتاب
- ⚖️ **جدول‌های تمایز** مفاهیم شبیه‌به‌هم + هشدارهای **«دام کنکور»**
- 🔎 **جدول همهٔ مثال‌ها ← کدام رفتار؟**
- 🧑‍🔬 **جدول دانشمندان** (نوبل ۱۹۷۳ و ۱۹۰۴)
- 🃏 **۲۲ فلش‌کارت تعاملی** (کلیک کن تا برگردد)
- 📝 **پاسخ نمونهٔ فعالیت‌های ۱ تا ۶**
- 🌙 **حالت تیره / روشن**
- ✏️🖍️🧹 ابزار **قلم، هایلایت و پاک‌کن** برای حاشیه‌نویسی روی صفحه

## 🚀 استفاده

فقط فایل `index.html` را در مرورگر باز کن. هیچ نصب یا اینترنتی لازم نیست (فقط برای بارگذاری فونت‌ها به اینترنت نیاز است).
