# تمرین جای خالی — زیست‌شناسی ۳

یک سایت ایستا برای تمرین «جای خالی» از روی کتاب درسی. هر جمله‌ی کتاب با **حداقل ۳ جای خالی**
روی واژه‌های کلیدی نمایش داده می‌شود؛ پاسخ می‌دهی و بلافاصله درست/غلط را می‌بینی.

بدون نیاز به سرور کار می‌کند: هم با دابل‌کلیک روی `index.html`، هم روی **GitHub Pages**.

## ساختار
```
build.py     ← سازنده: از PDF فایل data.js می‌سازد (روی کامپیوتر خودت)
index.html   ← صفحه‌ی سایت
style.css    ← ظاهر
app.js       ← منطق تمرین
data.js      ← محتوای تولیدشده (با build.py ساخته می‌شود)
```

## ۱) ساخت محتوا از روی PDF
```bash
pip install pymupdf
python3 build.py "Zist 3.pdf"
```
خروجی: `data.js`. اگر خواستی حداقلِ جای خالی را عوض کنی، در ابتدای `build.py`
مقدار `MIN_BLANKS` را تغییر بده.

## ۲) اجرای محلی (بدون سرور)
کافی است `index.html` را در مرورگر باز کنی. برای اجرای تحت وب محلی:
```bash
python3 -m http.server 4173
# سپس در مرورگر: http://localhost:4173
```

## ۳) بارگذاری روی GitHub
```bash
git init
git add .
git commit -m "سایت تمرین جای خالی زیست ۳"
git branch -M main
git remote add origin https://github.com/<USERNAME>/<REPO>.git
git push -u origin main
```
سپس در گیت‌هاب: **Settings → Pages → Branch: main / root → Save**.
چند دقیقه بعد آدرس `https://<USERNAME>.github.io/<REPO>/` فعال می‌شود.

> نکته: چون محتوا از کتاب درسی گرفته شده، اگر می‌خواهی خصوصی بماند مخزن را
> **Private** بگذار و سایت را فقط به‌صورت محلی (باز کردن `index.html`) استفاده کن.
> GitHub Pages برای مخزن خصوصی نیاز به پلن پولی دارد؛ برای مخزن عمومی رایگان است.

## کلیدهای میان‌بر
- `Enter`: رفتن به جای خالی بعدی / بررسی صفحه
- `→` و `←`: صفحه‌ی قبل و بعد
- دکمه‌ی 🌙 : حالت شب/روز
